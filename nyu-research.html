Computational Pipeline for Cancer Neoantigen Discovery

NYU Grossman School of Medicine — Summer Undergraduate Research Program (2025)

Overview

During my summer at NYU Grossman, I worked on a computational pipeline focused on identifying cancer-relevant neoantigen candidates from mutation-driven peptide changes. My work emphasized building reproducible data processing steps, generating peptide libraries, and preparing outputs for downstream immunology-focused filtering and prioritization.

Core Question

Can we systematically simulate and prioritize mutation-derived peptides that could be presented on MHC class I and serve as actionable therapeutic targets?

What I Built

  • Mutation-to-peptide generation: converted mutation tables into mutant protein sequences and overlapping 8–11mer peptides.
  • Wild-type comparisons: reconstructed WT counterparts to distinguish “truly novel” peptides vs unchanged sequence windows.
  • Filtering-ready outputs: formatted peptide tables for prediction tools (binding + prioritization workflows).
  • Reproducible organization: consistent file structure, naming, and run logs to support scaling + re-runs.

Methods & Tools

  • Python (pandas, Biopython), shell scripting
  • HPC workflows (job scheduling, large file I/O, batch runs)
  • Peptide generation logic (overlapping windows around mutation sites)
  • Downstream compatibility with prediction tools (e.g., binding / presentation scoring)

Note: I’m keeping some details general on purpose — I can expand this later with more specific figures/results once you decide what you want public.

Outputs

  • Cleaned peptide libraries suitable for prediction pipelines
  • Traceable mapping between mutation → protein → peptide windows
  • Tables structured for quick filtering and comparison

What I Learned

  • How small assumptions (reference choice, mapping rules, windowing) can bias results
  • How to design pipelines so they’re testable, debuggable, and scalable
  • How to communicate computational results clearly to a wet-lab / translational audience
